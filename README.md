# 小优在线考试系统 v2.0 开发版

## 🆕 v2.0 新增功能

### 🤝 好友管理系统
- **班级成员添加** - 在同班同学中快速添加好友
- **ID搜索添加** - 通过用户ID精确查找并添加好友
- **好友申请管理** - 完整的申请/审核/通过流程

### 💬 聊天通信系统
- **基于好友的私聊** - 只有好友间才能进行私聊
- **群聊创建管理** - 创建群聊、邀请成员、审核申请
- **统一消息系统** - 私聊和群聊的统一消息处理

### 📚 课程管理升级
- **选课申请审核** - 学生申请选课，管理员审核通过
- **课程通知发布** - 教师发布课程相关通知，支持置顶
- **作业管理系统** - 作业发布、提交、批改完整流程
- **选课管理员** - 专门的选课管理员角色和权限

### 🎨 界面重构
- **QQ风格三栏布局** - 左侧导航、中间列表、右侧详情
- **现代化UI设计** - 更加美观和符合现代审美的界面
- **响应式交互** - 流畅的交互体验和动画效果

## ⚙️ 开发环境配置

### 环境要求
- Qt 6.9.1+ (包含Network模块)
- MySQL 8.0+
- CMake 3.16+
- Git 2.0+

### 构建步骤
```bash
# 1. 克隆开发分支
git clone -b v2.0-dev https://github.com/yourusername/xiaoyou-exam-system.git

# 2. 安装依赖
# 确保Qt Network模块已安装

# 3. 数据库配置
# 导入v2.0扩展表结构
source db2/table_create_2.sql

# 4. 构建项目
mkdir build && cd build
cmake ..
cmake --build . --config Debug
```
